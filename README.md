# MFDP_ts_forecasting

Учебный проект MFDP 2026. Time series forecasting на открытых данных M5 Walmart.

Четыре задачи на одном датасете:

- иерархический прогноз продаж item × store на 28 дней;
- оценка эластичности спроса по цене с доверительными интервалами;
- оценка uplift промо-акций для планирования с понятным ROI;
- оптимизация цен и промо под бизнес-цель (выручка / маржа) с ограничениями.
